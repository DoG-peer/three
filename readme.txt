Three.jsの練習に作ったものです

sizeとarを変更すればべつのものもかけます．

ただしsizeの値は自然数で，
arはsize個の整数の配列をsize個並べた配列．
つまりsize×sizeの行列（二次元配列）を入れてください．

ar[i][j]の値は各iについてar[i][j]≠0になるようなjがちょうど二つだけあり、
i,jを変えて、
ar[i][j]の値は各jについてもar[i][j]≠0になるようなiがちょうど二つだけあるようなものを設定してください

はthree.js関連のファイルを公式サイトからダウンロードしたものの中に含まれています．
